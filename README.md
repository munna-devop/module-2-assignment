\### Live URL



https://todo-app-deployment-oxbt.onrender.com



\### How to Run Locally



1\. Clone the repository

2\. Create a virtual environment

3\. Install dependencies:

&nbsp;  pip install -r requirements.txt

4\. Run:

&nbsp;  python manage.py runserver
